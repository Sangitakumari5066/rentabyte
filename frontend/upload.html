How Your Files Are Protected

  1. AES Encryption (Fernet)
    Files are encrypted with a 256-bit key before leaving the server
  2. 5 MB Chunking
    Encrypted data is split — no single seller has your full file
  3. SHA-256 Integrity Check
    Each chunk has a hash. Download fails if any chunk is tampered
  4. Distributed Storage
    Chunks spread across multiple seller Dropbox accounts
